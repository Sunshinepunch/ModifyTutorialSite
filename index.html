Welcome to a tutorial site where you can make multiple changes to the DOM via JS functions

This handy little site will help anyone wanting to see basic dom interactions in action.

If you follow along with the Readme file included with github, you can build something yourself for practice!

List of Modifications

  1. Input field to update a text box
  2. Click on an image to open a browser page to a different URL
  3. Click a button to increment a counter
